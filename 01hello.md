整数类型： 分为三种
    整数类型：整数、浮点、定点                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           O                                                                                                                                                                        O 
        整数：分为有符号和无符号，无符号是0开始，有符号是负数开始。
            int -21E到+21E           ---------- 用这个
            bigint：-92EE到+92EE
        浮点：float和double 
            3.4e-38表示3.4×10^(-38)
        定数：
            decimal(5,2)  总长是5，小数点后占2位。 例如：123.45
    日期类型
        DATE       YYYY-MM-DD 格式    ---------- 用这个
        DATETIME   YYYY-MM-DD HH:mm:SS格式    ---------- 用这个
    字符串
        char    初始长度为固定长度
        varchar  长度可变   ---------- 用这个
        text     存很长的文本，文章等。 ---------- 用这个
